# Documentation

- `api/` - API references
- `deployment/` - deployment and hosting guides
- `i18n/` - multilingual implementation notes and translation references
- `plans/` - implementation plans
- `testing/` - test guides and QA notes
- `ui/` - UI-specific change summaries
